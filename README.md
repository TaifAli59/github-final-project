# Simple Interest Calculator

## Project Description
This repository contains a Bash script that calculates simple interest 
based on user input.

## Simple Interest Formula
Simple Interest (SI) = (Principal × Rate × Time) / 100

Where:
- **Principal (P)**: The initial amount of money
- **Rate (R)**: Annual interest rate (in percentage)
- **Time (T)**: Time period (in years)

## Example
- Principal = 1000
- Rate = 5%
- Time = 3 years
- Simple Interest = (1000 × 5 × 3) / 100 = **150**

## Usage Instructions
1. Clone the repository
2. Run the script:
```bash
bash simple-interest.sh
```
3. Enter the required inputs when prompted:
   - Principal amount
   - Rate of interest
   - Time period
4. The script will display the calculated Simple Interest
